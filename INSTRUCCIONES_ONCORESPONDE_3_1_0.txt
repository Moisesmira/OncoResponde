ONCORESPONDE 3.1.0 · MI SEGUIMIENTO

Cambios:
- Mi seguimiento reorganizado en: estado de hoy, síntomas, evolución, diario y resumen semanal.
- Escala específica del dolor de 0 a 10.
- Nuevos síntomas: falta de aire, tos y hormigueos.
- Resumen semanal de estado emocional, dolor medio, fatiga y sueño.
- Avisos clínicos claros, incluido el texto consensuado para fiebre de 38 °C o superior.
- Datos almacenados localmente en el dispositivo.

Compilación: npm ci && npm run build
