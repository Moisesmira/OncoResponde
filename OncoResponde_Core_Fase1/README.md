# OncoResponde Core · Fase 1

Proyecto real React + TypeScript + Vite + PWA + Netlify Functions.

## Subir a GitHub
Sube todo el contenido de esta carpeta a `Moisesmira/OncoResponde`.

## Configurar Netlify
- Build command: `npm run build`
- Publish directory: `dist`
- Variable de entorno: `OPENAI_API_KEY`
- Variable opcional: `OPENAI_MODEL` (por defecto `gpt-5-mini`)

## Desarrollo local
```bash
npm install
npm run dev
```
